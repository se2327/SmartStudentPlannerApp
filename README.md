SmartStudentPlanner is an Android app built in Kotlin to help students efficiently organize their daily tasks, study schedules, and important deadlines.

Features:-
1.Login & Register: Secure access for each student
2.Assignments: Add, delete, and track tasks easily
3.Exam Planner: Track preparation progress (how much completed, how much pending)
4.Profile Setup: Update personal info and add links to LinkedIn/GitHub
5.Feedback & Logout: Provide feedback and safely log out


Technologies Used:-
1.Kotlin
2.Android Studio
3.XML for UI design
4.Firebase (for storage/authentication)
